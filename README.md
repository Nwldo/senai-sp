## 📦️ Como rodar o projeto
Clone o projeto com o comando abaixo:
```bash
  # Instale o Node.js
  # Instale o Angular: Npm install –g @angular/cli
  # Clone o repositório
  ❯ git clone 
https://github.com/[usuario]/[nome-projeto].git
	# Entre no diretório
  ❯ cd [nome-projeto]
	# Execute o projeto
  ❯ dotnet run
```
